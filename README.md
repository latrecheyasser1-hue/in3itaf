# انعطاف (in3itaf)

Luxury jewelry and Islamic art platform for an Algerian artist.

## Tech Stack
- **Frontend**: React, Tailwind CSS, Framer Motion
- **Backend**: Supabase (Database & Auth)
- **State Management**: React Context API
- **Icons**: Lucide React

## Dashboard
Includes a full dynamic CMS to manage:
- Site settings (titles, description, social links)
- Artwork gallery
- Fashion categories
- Book presentation
- Customer orders
